# 큐란?

큐는 스택과 다르게 먼저 들어간 데이터가 먼저 나오는 방식이다.
해당 개념을 이용하여 C의 배열을 사용하여 큐 자료구조를 만들어보도록 하겠다.

		printf("값이 없음");
		exit(1);
	}

		queue->cur = 0;

	queue->numOfQueue--;

	return queue->queue[queue->cur];
